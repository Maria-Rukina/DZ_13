# Домашнее задание к работе 13

Условие задачи

Составить программу, которая во введенной пользователем строке меняет местами первое и последнее слово во введенной строке.

----

1.Алгоритм и блок схема

Алгоритм

1. Начало

2. Вводим массив для строки char str[100];

3. Вводим строку с помощью функции fgets(str, sizeof(str), stdin); которая позволяет ввести имменно слова, которые она понимает, а не рад знаков

4. Используем str[strcspn(str, "\n")] = '\0'; чтобы убрать конец строки \n

5. Вводим новый массив char* words[50]; для слов и int n = 0; для счёта кол-во слов

6. Вводим char* token = strtok(str, " \t"); для разделения слов по пробелам

7. С помощью while (token != NULL && n < 50) перебираем слова и записываем их в words[n++] = token;

8. Теперь создаём пустой указатель и записываем туда первое слово char* temp = words[0];

9. В это первое слово записываем последнее слово words[0] = words[n - 1];

10. И в последнее слово вставляем первое  words[n - 1] = temp;

11. С помощью for (int i = 0; i < n; i++) выводим строку
